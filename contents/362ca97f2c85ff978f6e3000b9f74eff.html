Acesso ao sistema

UMLUseCase
Untitled :: Model1 :: Acesso ao sistema

Description

FLUXO PRINCIPAL

  1. Usuário acessa área de administração de carteira.
  2. Usuário visualiza lista de ações presentes na carteira. [FA1, FA2] [FA3] [FA4]
  3. Usuário pode visualizar saldo presente na carteira.
  4. Usuário visualiza estatísticas do seu patrimônio
  5. Usuário solicita relatório do período filtrado.

FA1

2.1 Usuário seleciona aba de Histórico de Ordens.
2.2 Usuário visualiza ativos e tipos de ordem (venda ou compra).
2.3 Usuário seleciona ativo para ver mais detalhes.

FA2

2.4 Usuário acessa aba de ativos em carteira.
2.5 Usuário seleciona “Criar Grupo de Ações”
2.6 Usuário digita nome do grupo.
2.7 Usuário escolhe ações que entrarão para o grupo.
2.8 Visualiza grupos.

FA3

2.9 Seleciona botão de ordem de compra/venda rápida das ações em carteira.
2.10 Usuário é direcionado à aba de ordem de compra/venda.

FA4

2.11 Usuário seleciona aba de histórico de acesso ao login.
2.12 Usuário visualiza data e horário dos acessos.
